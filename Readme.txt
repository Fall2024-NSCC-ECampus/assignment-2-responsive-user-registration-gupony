****Responsive Registration Project****

This is a web application where users can register by entering a username, email, and password. The password gets securely hashed before being saved to the database.

**Technologies Used**

Spring Boot
Thymeleaf
MySQL
BCrypt
Bootstrap

**Project Structure**

controllers: Manages the registration process.

models: Defines the user data.

repositories: Handles database interactions.

templates: Contains the registration form (HTML).
